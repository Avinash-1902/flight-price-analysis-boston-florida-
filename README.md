# Flight Price Analysis: Boston to Florida

This project analyzes flight prices from Boston (BOS) to major Florida airports for a selected departure date.

## Objective
Compare airfare across airlines and destinations:
- MIA
- MCO
- TPA
- FLL

## Tools Used
- Python
- Playwright
- Pandas
- Power BI

## Project Workflow
1. Scrape flight data from Skiplagged
2. Clean and filter the dataset
3. Build an interactive Power BI dashboard

## Dashboard Insights
- Cheapest ticket
- Average fare
- Airline fare comparison
- Destination fare comparison
- Price vs number of stops
